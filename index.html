âš™ï¸ How It Works

The script automates the entire router reboot process using Selenium WebDriver:

1 Login: Opens your router's web interface and logs in with your credentials
2 Navigate: Accesses the Advanced settings menu
3 Locate: Finds the Management section and Reboot option
4 Execute: Clicks the reboot button and confirms the action
5 Monitor: Detects when internet connection is lost (indicating successful reboot)

ğŸš€ Quick Start

Prerequisites

  • Python 3.6 or higher
  • Chrome, Chromium, or Brave browser
  • ChromeDriver (matching your browser version)

Installation

1. Clone the repository:

git clone https://github.com/ibrahim24-crypto/cyberrouter_reboot_improved.py.git

2. Install dependencies:

pip install selenium

3. Download ChromeDriver:

Visit chromedriver.chromium.org and download the version matching your browser

4. Configure the script:

Edit the configuration section at the top of the script with your router details

ğŸ” Element Selectors (For Different Router Brands)

Use your browser's Developer Tools (F12) to find the correct selectors for your router:

  • LOGIN_USERNAME_ID - ID of the username input field
  • LOGIN_PASSWORD_ID - ID of the password input field
  • MANAGEMENT_TAB_XPATH - XPath to the Management menu
  • REBOOT_BUTTON_XPATH - XPath to the Reboot button
ğŸ’¡ Tip: Open your router's web interface, right-click on elements, select "Inspect", and copy the element's ID or XPath to use in the configuration.

â–¶ï¸ Usage

Run the script:

python3 rebootrouter.py

Select your preferred mode:

  • Press 1 for Normal Mode
  • Press 2 for Hacker Mode
  • Press 0 to exit

The script will then automatically:

  1. Open your browser
  2. Log into your router
  3. Navigate to the reboot section
  4. Execute the reboot
  5. Monitor for connection loss
  6. Exit when reboot is confirmed
